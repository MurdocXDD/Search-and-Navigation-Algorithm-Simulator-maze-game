# Search-and-Navigation-Algorithm-Simulator-maze-game
Juego de Laberinto Automático Este es un juego de laberinto interactivo desarrollado en Python con la librería tkinter. El propósito principal es demostrar la generación de un laberinto con elementos dinámicos y, sobre todo, su resolución automática mediante un algoritmo de búsqueda de fuerza bruta.
